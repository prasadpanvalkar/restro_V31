## WebSocket Communication Flow

```
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|  Django Server |     |  Redis        |     |  Client        |
|                |     |                |     |                |
+-------+--------+     +-------+--------+     +-------+--------+
        |                      |                      |
        | Connect WebSocket    |                      |
        |<---------------------------------------------+
        |                      |                      |
        | Join Group           |                      |
        +--------------------->|                      |
        |                      |                      |
        | Event Occurs         |                      |
        | (e.g., New Order)    |                      |
        |                      |                      |
        | Send to Group        |                      |
        +--------------------->|                      |
        |                      |                      |
        |                      | Broadcast to         |
        |                      | Group Members        |
        |                      +--------------------->|
        |                      |                      |
        |                      |                      | Handle Event
        |                      |                      | Update UI
        |                      |                      v
        |                      |             +-------+--------+
        |                      |             |                |
        |                      |             | UI Updated     |
        |                      |             |                |
        |                      |             +----------------+
```

## User Authentication Flow

```
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|  Client        |     |  Django Server |     |  Database      |
|                |     |                |     |                |
+-------+--------+     +-------+--------+     +-------+--------+
        |                      |                      |
        | Login Request        |                      |
        +--------------------->|                      |
        |                      |                      |
        |                      | Verify Credentials   |
        |                      +--------------------->|
        |                      |                      |
        |                      |<---------------------+
        |                      |                      |
        |                      | Generate JWT Token   |
        |                      |                      |
        |<---------------------+                      |
        | Return Token         |                      |
        |                      |                      |
        | API Request with     |                      |
        | Authorization Header |                      |
        +--------------------->|                      |
        |                      |                      |
        |                      | Verify Token         |
        |                      |                      |
        |                      | Process Request      |
        |                      |                      |
        |<---------------------+                      |
        | Return Response      |                      |
        |                      |                      |
```
